ip4STATICS: interactive problems for STATICS

Use ip4STATICS to build skill and confidence in your STATICS know-how.


Each problem has the following action buttons:
INSTANCE, ENTER, ANS, WORKSHEET, SOLUTION.

Using INSTANCE to generate new instance variables

Using ENTER to enter answers and get feedback

Using ANS to call for an instance-specific answer

Using WORKSHEET to get a worksheet
A WORKSHEET is available for every problem. The idea is to print it and work your solution on it.

Using SOLUTION to get a problem solution
A SOLUTION is available for every problem. The idea is to use it as a validation if you feel you need it.
