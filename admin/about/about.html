About app4STATICS

Using this active problem pool for STATICS to build your skills and test yourself!


What is 'app4STATICS'?
app4STATICS is a rich collection of active problems for building and verifying your STATICS skills.

What is an active problem?

An active problem has a set of parameters which vary with each instance. Parameters are chosen randomly within ranges. In this way you can try as many variations of the the same problem type as you wish.


What about active problem answers?

Each problem instance has its answer set generated immediately. Then you can enter your answers and get immediate feedback.


What do I need to know to use 'app4STATICS' effectively?
Use app4STATICS while you are learning the principles and methods of Statics to earn how to apply them. And use app4STATICS at any time to review and refresh your skills. Use app4STATICS to assess for yourself how competent you are in Statics problem-solving.

How are the problems organized?
The problems are organized by units which match the most commonly used topics or chapters in popular texts and videos.

Who controls the use of app4STATICS?
YOU ARE IN COMPLETE CONTROL OF HOW YOU USE app4STATICS.
(1) You get immediate feedback when you enter an answer.
(2) You can ask for an answer at any time.
(3) You can ask for a worked solution at any time.
(4) You can ask for a new set of instance variables, such as loads, lengths, masses, etc. By running multiple instances of a problem you can understand the structure of the problem type, not just how to solve for a particular set of numbers.
